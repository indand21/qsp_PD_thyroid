# v1 Pipeline Results (IN-SILICO)

> **These outputs are model-generated, not real patient data.** They *execute*
> the objectives described in "Manuscript v.1" — a mechanistic QSP model with
> CTLA-4/combination therapy, biphasic dynamics, autoantibody seroconversion,
> global sensitivity analysis, virtual population, validation cohorts, and an ML
> ensemble. They are in-silico results, not empirical clinical findings.

**Calibration.** The expanded model (`qsp_expanded/`) is CALIBRATED so that
180-day per-drug hypothyroidism incidence lands in published ranges with correct
class ordering (Combination > PD-1 > PD-L1 > CTLA-4) and 60-140 day onset — see
`calibration_report.json`. The **epidemiological** outputs (incidence by class,
virtual population, sensitivity) reflect this calibrated population.

**Cohorts are covariate-sampled.** The `cohort_*.csv` used for the ML /
validation / odds-ratio stages sample each patient's susceptibility from their
risk factors (age, sex, HLA-DQ, baseline TSH) plus a random effect, giving a
12-month event rate (~12-19%) modestly higher than the marginal calibrated
population incidence (mild covariate risk enrichment). This is noted in the
cohort generator docstring.

Generated by `scripts/run_v1_pipeline.py` at the scale in
`config/v1_pipeline.yaml`.

## Reproduce

```
python scripts/run_v1_pipeline.py
```

Edit `config/v1_pipeline.yaml` to change scale (default is reduced-but-real;
full-scale magnitudes are noted inline in that file).
